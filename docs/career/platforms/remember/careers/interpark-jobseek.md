# 인터파크(놀유니버스) - 리멤버 경력 기술 (구직용/상세)

> **회사**: (주)인터파크 (놀유니버스)
> **부서**: 쇼핑 개발팀
> **직급**: 백엔드 개발자
> **기간**: 2021.09 - 2022.01 (5개월)
> **용도**: 적극 구직 의사 표현
> **톤**: 성과 강조, 적극적 톤

---

## 경력 내용

```
[레거시 시스템 TDD 도입 및 테스트 문화 정착]
레거시 시스템에 테스트 코드가 전무하여 기능 변경 시 사이드 이펙트 파악 불가, 판매자 정산/주문 조회 등 핵심 기능의 정합성 검증 수단 부재, 코드 수정 시 수동 테스트에 의존하여 배포 리스크 증가
→ 신규 기능 개발부터 TDD 방식 적용, 테스트 우선 개발 문화 정착
→ 핵심 비즈니스 로직(Seller 아이템 조회, 특정 셀러의 주문 관련 통합 데이터 조회)에 단위 테스트 작성
→ 테스트 가능한 구조로 복잡한 비즈니스 로직을 Service Layer로 분리하여 코드 가독성 향상
→ 팀 내 테스트 작성 가이드 공유 및 코드 리뷰 시 테스트 코드 포함 여부 검토 체계 도입

[성과]
- 테스트 기반 회귀 검증으로 배포 리스크 감소
- 배포 후 버그·핫픽스 건수 감소
- 수동 테스트 시간 절감으로 개발 생산성 향상
- 테스트 문화 정착으로 코드 변경에 대한 신뢰성 확보
```
