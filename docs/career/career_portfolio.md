> **한 줄 요약**: Elasticsearch 기반 자체 검색 엔진 구축으로 라이선스 비용 절감 및 데이터 등록 속도 70% 단축

### 배경/문제 정의 (Why)
- 상용 검색 솔루션의 높은 라이선스 비용 부담
- 대량 기록물 이미지 등록 시 단건 Insert로 처리 지연 발생

### 해결 전략 (How)
- Elasticsearch 기반 자체 검색 엔진 개발, Nori 한글 형태소 분석기 적용
- 단건 Insert → Bulk Insert 전환으로 DB 트랜잭션 횟수 최소화

### 성과 (Result)

| 항목 | Before | After | 개선율 |
|------|--------|-------|--------|
| 데이터 등록 속도 | 기준 | - | **70% 단축** |
| 라이선스 비용 | 발생 | 제거 | **비용 절감** |
