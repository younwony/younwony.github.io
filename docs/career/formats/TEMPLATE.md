---

## 파일명 규칙

```
by-jd/{company}_{position}.md      # 예: toss_backend.md
by-company/{type}.md                # 예: startup.md, enterprise.md

```

## 프로젝트별 강조 키워드

### 인플루언서 데이터 플랫폼
- **대용량**: 200만 건, 검색 10배 개선 (10초+→1초 이내)
- **자동화**: 수작업→일 5,000건 자동 수집, Zero Ops
- **확장성**: 10만→200만 데이터 20배 확대

### Dynamic Pricing
- **정산/가격**: 마진율 4% 보장 자동 가격 조정
- **매출 임팩트**: 매출/트래픽 각각 10% 상승
- **배치 처리**: Spring Batch 대량 안정 처리

### 이미지 정합성
- **데이터 정합성**: SHA256 해시 검증, 클레임 Zero
- **무결성**: URL 동일해도 변경 감지

### ChatOps
- **자동화**: 운영 업무 40%→0% (Zero Ops)
- **생산성**: 버튼 클릭으로 업무 완결
- **협업**: 채널 100% 통합

### AI 기반 생산성 혁신
- **개발 속도**: 운영 업무 90% 단축 (1~2일→1~2시간)
- **팀 문화**: 코드 리뷰 문화 정착, 컨벤션 수립
- **AI 활용**: Claude Code 페어프로그래밍
