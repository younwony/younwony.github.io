# Settle - Alibaba 외부몰 주문 연동

> 작성일: 2025-12-31
> 소속: (주)구하다 / Tech 팀
> 기간: 2023.08 - 2023.11

---

## 개요

Alibaba 주문건에 대한 관리 페이지 구축 및 API 연동 프로젝트. 주문 접수, 요청, 취소 등 전 과정을 연동하고 History 관리 기능 구현.

---

## 배경 및 문제점

### 기존 상황
- Alibaba 주문에 대한 관리 시스템 부재
- 주문 상태 추적 어려움

### 문제점
- **관리 어려움**: Alibaba 주문에 대한 통합 관리 페이지 필요
- **히스토리 부재**: 주문 이력 추적 불가

### 목표
- Alibaba 주문 관리 페이지 구축
- 주문 접수/요청/취소 API 연동
- 주문 History 관리 기능 구현

#### 1. Alibaba API 연동

#### 2. 주문 History 구축
- Alibaba 주문 이력 저장 및 조회 기능

#### 3. 관리 페이지
- Settle 내 Alibaba 주문 관리 페이지 구현
