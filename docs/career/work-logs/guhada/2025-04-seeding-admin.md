# Seeding Admin 구축

> 작성일: 2025-12-31
> 소속: (주)구하다 / Tech 팀
> 기간: 2025.04 - 2025.06

## 개요

Seeding Admin 구축 프로젝트. Exolyt → Webscraper, TikTok API 연동 등 데이터 수집 프로세스부터 Admin 시스템 구축, ElasticSearch 기반 검색 필터 최적화까지 전 과정을 주도적으로 설계하고 개발.

---

## 배경 및 문제점

### 기존 상황
- Seeding 관련 업무 수작업으로 진행
- 데이터 수집 및 관리 체계 부재

### 문제점
- **수작업 비효율**: 담당자의 반복적인 수작업
- **데이터 관리 어려움**: 체계적인 데이터 관리 부재

### 목표
- 데이터 수집 자동화
- Admin 시스템 구축
- 업무 효율성 향상

| 구분 | 기술 |

#### 2. Admin 시스템 구축
- Seeding 관리용 Admin 시스템 개발
- 데이터 관리 기능 구현

#### 3. ElasticSearch 기반 검색 필터 최적화
- 검색 필터 기능 구현
- 검색 성능 최적화
