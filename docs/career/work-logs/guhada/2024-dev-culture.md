# 개발 문화 개선 및 AI 기반 팀 생산성 혁신

> 작성일: 2025-12-24
> 소속: (주)구하다 / Tech 팀
> 기간: 2024 - 진행 중

---

## 개요

코드 리뷰 문화 정착 및 AI 에이전트(Claude Code) 활용으로 팀 생산성을 혁신한 프로젝트.

---

## 배경 및 문제점

### 기존 상황
- 팀 내 통일된 코드 컨벤션과 리뷰 프로세스 부재
- 코드 리뷰 없이 배포 진행
- 월간/일간 운영성 반복 업무에 시니어 개발자 리소스 과다 투입

### 문제점
- **품질 저하**: 개발자마다 스타일이 상이, 협업 비용 증가
- **운영 장애**: 코드 리뷰 없이 배포되어 사이드 이펙트 반복 발생
- **리소스 낭비**: 로그 조회, 데이터 추출, 어드민 기능 개발에 시니어 리소스 투입

### 목표
- 건강한 코드 리뷰 문화와 품질 기준 정착으로 운영 장애 사전 차단
- AI 에이전트 기반 자동화로 반복 업무 흡수
- 팀 전체 기술 역량과 생산성 상향 평준화

### 구현 내용

#### 1. 코드 리뷰 문화 정착
- PR 기반 리뷰 프로세스 제안 및 도입
- 배포 전 최소 2인 승인 체계 확립
- 설계·성능·운영 요소를 함께 논의하는 "더 나은 설계" 중심 리뷰 문화 형성

#### 2. 코딩 컨벤션 수립
- Java/Spring 코딩 컨벤션을 팀원들과 함께 정의하고 문서화
- 신규 입사자 온보딩 시 즉시 적용 가능하도록 체계화

#### 3. AI 에이전트 활용 자동화
- Claude Code를 활용하여 SQL 작성, 데이터 조회·정제, 운영성 스크립트를 AI 페어프로그래밍으로 신속 구현
- 기존 1~2일 소요 작업을 1~2시간으로 단축

#### 4. MCP 기반 협업 자동화
- Confluence MCP 연동으로 작업 완료 시 기술 문서 자동 생성
- 백엔드 팀 내 기술적 의사결정 히스토리 축적
- 프론트엔드 개발자 API 연동 시간 단축 (API 문의 80% 감소)

#### 5. 운영 툴 내재화
- 반복 업무를 내부 시스템 기능으로 내재화
- 사내 맞춤형 운영 툴 5건 이상 신규 구축

#### 6. 팀 내 AI 활용 전파
- 프롬프트 엔지니어링 노하우와 AI 활용 패턴 팀 내 공유
- 주니어 개발자도 복잡한 쿼리 및 테스트 코드 작성에 AI 보조 활용

### 기술적 의사결정

| 결정 | 선택 | 이유 |
|------|------|------|
| AI Agent | Claude Code | 자연어 명령으로 SQL/스크립트 생성, 반복 업무 90% 시간 단축 및 팀 전체 활용 가능 |
| 문서화 자동화 | Confluence MCP | 사내 표준 문서 도구, 작업 완료 즉시 문서화로 정보 정확도 향상 |

## 결과 및 성과

### 정량적 성과
- **코드 품질 개선**: 배포 후 버그·핫픽스 건수 감소, 운영 장애 사전 차단
- **온보딩 속도 단축**: 일관된 컨벤션과 리뷰 프로세스로 신규 입사자 생산 투입 시점 앞당김
- **운영 업무 90% 단축**: 운영성 도구 개발 시간 1~2일 → 1~2시간으로 개선
- **팀 역량 향상**: 리뷰 기반 학습·지식 공유로 주니어 개발자 역량 상승
- **API 문의 80% 감소**: MCP 기반 문서 자동화로 프론트엔드 협업 효율 향상

### 정성적 효과
- 시니어 개발자가 반복 작업에서 해방되어 핵심 도메인 개발·아키텍처 작업에 집중
- 개발팀 전반의 협업 효율·코드 품질·개발 속도 유의미 개선
