# 해외 부티크 API 연동

> 작성일: 2025-12-31
> 소속: (주)구하다 / Tech 팀
> 기간: ~ 2025.01

---

## 개요

Netdressed, Rakuten 등 해외 부티크의 API 연동 작업. 주도적으로 설계와 개발을 진행하고, Product-Engine 상품 등록 프로세스에 대한 이해도를 높이며 부분적인 리팩토링으로 성능과 코드 품질을 향상시킨 프로젝트.

---

## 배경 및 문제점

### 기존 상황
- 해외 부티크 상품 등록 프로세스 존재
- 신규 부티크 연동 필요

### 문제점
- **기술 부채**: 기존 Product-Engine 코드의 기술 부채 존재
- **연동 복잡도**: 각 부티크별 API 스펙 상이

### 목표
- Netdressed, Rakuten 등 해외 부티크 API 연동
- Product-Engine 부분 리팩토링
- 코드 품질 향상
