# 상품 옵션 재고 알림 개발

> 작성일: 2025-12-31
> 소속: (주)구하다 / Tech 팀
> 기간: 2023.07 - 2023.09

---

## 개요

상품 옵션 단위로 품절 이후 재고 입고 시 Push 알림을 받을 수 있는 재고 입고 알림 기능 구현 프로젝트.

---

## 배경 및 문제점

### 기존 상황
- 품절 상품에 대한 재입고 알림 기능 부재
- 고객이 품절 상품 재입고 여부를 직접 확인해야 함

### 문제점
- **고객 이탈**: 품절 시 다른 플랫폼으로 이탈 가능성
- **재방문 유도 어려움**: 재입고 시 고객 알림 수단 부재

### 목표
- 상품 옵션 단위 재고 입고 알림 기능 구현
- 고객 만족도 향상

---

## 상세 구현

### 기술 스택

| 구분 | 기술 |
|------|------|
| 언어 | Java |
| 프레임워크 | Spring Boot |
| 데이터베이스 | MySQL |
| 푸시 | FCM (Firebase Cloud Messaging) |

### 구현 내용

#### 1. 재고 입고 알림 신청/조회 API
- 고객이 특정 상품 옵션에 대해 알림 신청 가능
- 알림 신청 내역 조회 기능 제공

#### 2. 재고 입고 알림 Batch
- 재고 입고 감지 Batch 구현
- 알림 대상 고객에게 Push 알림 발송

#### 3. Device Token Update
- Device Token 갱신 로직 구현
- 만료된 토큰 처리

### 기술적 의사결정

| 결정 | 선택 | 이유 |
|------|------|------|
| 알림 방식 | Push 알림 | 즉각적인 알림으로 재방문 유도 효과 극대화 |
| 처리 방식 | Batch | 재고 변경 시점 감지 및 일괄 처리 효율 |
