# 암호화폐 구하다 포인트 전환 개발

> 작성일: 2025-12-31
> 소속: (주)구하다 / Tech 팀
> 기간: 2023.11 - 2023.12

---

## 개요

Temco 코인으로 구하다 상품을 구매할 수 있도록 코인 → 포인트 전환 프로세스를 구축한 프로젝트.

---

## 배경 및 문제점

### 기존 상황
- Temco 코인 보유 고객이 구하다 플랫폼에서 상품 구매 불가
- 암호화폐 결제 수단 부재

### 문제점
- **결제 한계**: Temco 코인을 구하다 플랫폼에서 사용 불가
- **고객 편의성**: 코인 보유 고객의 구매 경로 제한

### 목표
- Temco 코인 → 구하다 포인트 전환 기능 구현
- 코인 보유 고객의 상품 구매 지원

---

## 상세 구현

### 기술 스택

| 구분 | 기술 |
|------|------|
| 언어 | Java |
| 프레임워크 | Spring Boot |
| 블록체인 | Temco |
| 데이터베이스 | MySQL |

### 구현 내용

#### 1. User 지갑 생성
- 사용자별 Temco 지갑 생성 기능 구현
- 지갑 주소 관리

#### 2. Temco 입금 처리
- Temco 코인 입금 감지
- 입금 내역 저장 및 관리

#### 3. Point 전환 프로세스
- Temco → Point 전환 로직 구현
- 전환 비율 적용 및 포인트 적립

### 기술적 의사결정

| 결정 | 선택 | 이유 |
|------|------|------|
| 전환 방식 | 자동 전환 | 사용자 편의성 극대화 |
