# Settle - 메인 팝업 관리 기능 구현

> 작성일: 2025-12-31
> 소속: (주)구하다 / Tech 팀
> 기간: 2023.06

---

## 개요

Guhada 구매 이용약관 고지를 위한 Main Popup 관리 기능을 Settle(Backoffice)에 빠르게 구현한 프로젝트.

---

## 배경 및 문제점

### 기존 상황
- 구매 이용약관 고지 Popup 필요
- Main Popup 관리 기능 부재

### 문제점
- **긴급 요구사항**: 이용약관 고지를 위한 Popup 빠른 구현 필요
- **관리 기능 부재**: Popup 관리 기능이 없어 하드코딩 필요

### 목표
- Main Popup 관리 기능 빠른 구현
- Settle 내에서 Popup 관리 가능하도록 구성

---

## 상세 구현

### 기술 스택

| 구분 | 기술 |
|------|------|
| 언어 | Java |
| 프레임워크 | Spring Boot |
| 데이터베이스 | MySQL |

### 구현 내용

#### 1. Main Popup 관리 기능
- Settle 내 Main Popup 관리 기능 구현
- Popup 생성/수정/삭제 기능 제공

#### 2. 빠른 구현
- 긴급 요구사항에 대응하여 빠른 개발 진행

### 기술적 의사결정

| 결정 | 선택 | 이유 |
|------|------|------|
| 구현 방식 | Settle 내 관리 기능 | 비개발자도 Popup 관리 가능 |
