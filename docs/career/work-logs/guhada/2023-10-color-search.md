# 상품 색상 검색 기능 개발

> 작성일: 2025-12-31
> 소속: (주)구하다 / Tech 팀
> 기간: 2023.10 - 2023.11

---

## 개요

상품에 대한 컬러 검색 기능 추가 프로젝트. 해외 상품 등록 시 color 값을 Guhada 플랫폼에서 검색 가능하도록 Color 표준화 알고리즘 구성 및 매핑 기능 구현.

---

## 배경 및 문제점

### 기존 상황
- 상품 검색 시 색상 필터 미제공
- 해외 부티끄별 Color 데이터 형식 상이

### 문제점
- **검색 한계**: 색상 기반 검색 불가 (예: "검은색 티셔츠")
- **데이터 불일치**: 해외 부티끄별 Color 표현 방식 상이

### 목표
- 상품 색상 검색 기능 추가
- Color 표준화 알고리즘 구성
- 해외 부티끄 Color 데이터 매핑

---

## 상세 구현

### 기술 스택

| 구분 | 기술 |
|------|------|
| 언어 | Java |
| 프레임워크 | Spring Boot |
| 검색 | Elasticsearch |
| 데이터베이스 | MySQL |

### 구현 내용

#### 1. Color 표준화 알고리즘
- 다양한 Color 표현을 표준 Color로 변환하는 알고리즘 구성
- 유사 색상 그룹핑

#### 2. 해외 부티끄별 Color 매핑
- 각 부티끄의 Color 데이터를 표준 Color로 매핑
- 매핑 테이블 구축

#### 3. 색상 검색 기능
- Elasticsearch 기반 색상 필터 검색 구현
- 상품 검색 시 색상 필터 제공

### 기술적 의사결정

| 결정 | 선택 | 이유 |
|------|------|------|
| 표준화 방식 | 알고리즘 기반 | 다양한 Color 표현에 유연하게 대응 가능 |
| 검색 엔진 | Elasticsearch | 기존 검색 인프라 활용 및 빠른 필터링 |
