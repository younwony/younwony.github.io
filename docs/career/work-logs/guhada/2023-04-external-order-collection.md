# ProductEngine - 외부 주문 수집 개발

> 작성일: 2025-12-31
> 소속: (주)구하다 / Tech 팀
> 기간: 2023.04 - 2023.05

---

## 개요

외부몰 주문 요청건에 대한 주문 수집 및 발주 확인 프로세스 구축 프로젝트.

---

## 배경 및 문제점

### 기존 상황
- 외부몰 주문에 대한 처리 프로세스 부재
- 외부몰 주문 요청 ~ 발주 확인까지 수동 처리

### 문제점
- **프로세스 부재**: 외부몰 주문 처리 자동화 필요
- **비효율성**: 수동 처리로 인한 시간 소요

### 목표
- 외부몰 주문 요청 ~ 발주 확인 프로세스 자동화
- ProductEngine 내 외부 주문 수집 기능 구현

---

## 상세 구현

### 기술 스택

| 구분 | 기술 |
|------|------|
| 언어 | Java |
| 프레임워크 | Spring Boot |
| 데이터베이스 | MySQL |

### 구현 내용

#### 1. 외부몰 주문 요청 처리
- 외부몰 주문 요청 수신 기능 구현
- 주문 데이터 정규화

#### 2. 내부 주문 생성
- 외부 주문 → 내부 주문 변환 로직 구현

#### 3. 발주 확인 프로세스
- 외부 주문 요청에 대한 발주 확인 프로세스 구축
- 상태 관리 및 히스토리 추적

### 기술적 의사결정

| 결정 | 선택 | 이유 |
|------|------|------|
| 프로세스 설계 | 요청→내부주문→발주확인 | 명확한 상태 관리 및 추적 가능 |
