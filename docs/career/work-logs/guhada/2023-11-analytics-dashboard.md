# Guhada Analytics 공유

> 작성일: 2025-12-31
> 소속: (주)구하다 / Tech 팀
> 기간: 2023.11 ~

---

## 개요

Guhada 플랫폼 내에서 이루어지는 가입, 주문, 클레임 등의 스탯을 집계하고 공유하는 Analytics 대시보드 구축 프로젝트.

---

## 배경 및 문제점

### 기존 상황
- 플랫폼 데이터 분석 및 공유 체계 부재
- 의사결정을 위한 데이터 접근 어려움

### 문제점
- **데이터 접근성**: 주요 지표 확인을 위해 DB 직접 조회 필요
- **분석 어려움**: 통합된 분석 도구 부재
- **의사결정 지연**: 데이터 기반 의사결정 어려움

### 목표
- 가입/주문/클레임 등 주요 스탯 집계
- 데이터 분석 및 공유 체계 구축
- 플랫폼 방향성 논의 가능하도록 지원

---

## 상세 구현

### 기술 스택

| 구분 | 기술 |
|------|------|
| 데이터 집계 | DB Data 집계 |
| 분석 도구 | BigQuery, GA (Google Analytics) |

### 구현 내용

#### 1. DB Data 집계
- 가입, 주문, 클레임 등 주요 지표 집계
- 일별/주별/월별 통계 산출

#### 2. BigQuery 활용
- 대용량 데이터 분석을 위한 BigQuery 활용
- 데이터 웨어하우스 구축

#### 3. GA 분석
- Google Analytics 연동 및 분석
- 사용자 행동 데이터 분석

### 기술적 의사결정

| 결정 | 선택 | 이유 |
|------|------|------|
| 분석 도구 | BigQuery + GA | 대용량 처리 및 사용자 행동 분석 가능 |
