# 메인 Curation 구현

> 작성일: 2025-12-31
> 소속: (주)구하다 / Tech 팀
> 기간: 2023.01 - 2023.02

---

## 개요

Guhada 플랫폼 메인 페이지 Curation 2차 변경 프로젝트. 객체 구조 설계를 통해 빠른 기능 확장이 가능하도록 구현.

---

## 배경 및 문제점

### 기존 상황
- 1차 Curation 시스템 운영 중
- 새로운 Curation 요구사항 발생

### 문제점
- 기존 구조에서 새로운 Curation 추가 시 복잡도 증가 우려

### 목표
- 2차 Curation 추가
- 향후 확장 가능한 구조 설계

---

## 상세 구현

### 기술 스택

| 구분 | 기술 |
|------|------|
| 언어 | Java |
| 프레임워크 | Spring Boot |
| 데이터베이스 | MySQL |

### 구현 내용

#### 1. 객체 구조 설계
- 초기 Curation 설계 시 객체 구조 설계 적용
- 확장성을 고려한 인터페이스 설계

#### 2. Curation Item 추가
- 2차 Curation 테스크에서 빠른 Curation Item 추가 구현
- 객체 구조 설계로 인한 개발 속도 향상

### 기술적 의사결정

| 결정 | 선택 | 이유 |
|------|------|------|
| 설계 방식 | 객체 구조 설계 | 향후 Curation 확장 시 빠른 개발 가능 |

### 정량적 성과
- 객체 구조 설계로 2차 Curation 빠른 추가 완료

### 정성적 효과
- 확장 가능한 구조로 향후 Curation 추가 용이
