# Settle - 발주확인 기능 개발

> 작성일: 2025-12-31
> 소속: (주)구하다 / Tech 팀
> 기간: 2023.03 - 2023.04

---

## 개요

Backoffice(Settle) 발주 확인 기능 구현. 발주 확인과 외부 주문 API 연동, 일괄 발주 확인 기능 개발.

---

## 배경 및 문제점

### 기존 상황
- 발주 확인 프로세스 수동 처리
- 외부 주문 연동 필요

### 문제점
- **수동 처리**: 발주 확인 작업의 수동 처리로 인한 비효율
- **외부 연동 부재**: 외부 몰 주문에 대한 발주 확인 연동 필요

### 목표
- Settle 내 발주 확인 기능 구현
- 외부 주문 API 연동
- 일괄 발주 확인 기능 제공
