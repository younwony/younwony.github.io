## 프로젝트 개요

개발자 부부로서 결혼 준비 중 개발과 관련된 컨텐츠를 만들어보고자 시작한 모바일 청첩장 프로젝트입니다. 프로젝트명 "ddangbbo"는 서로의 애칭인 '땅'과 '뽀'를 합쳐서 지었습니다.

### 기술 스택
- **Backend**: Spring Boot, MySQL, JPA
- **Frontend**: jQuery, JavaScript
- **DevOps**: GitHub Actions (CI/CD)
- **Infrastructure**: AWS EC2, Route 53

## 5. 프로젝트 개발

**작성일**: 2025.03.04

> "가장 많이 공들였던 개발 부분이다"

아내가 디자인 및 기타 측면을 지원했습니다.

### 구현된 기능 (10가지)

#### 1. Welcome Section
- 웨딩 사진 표시
- 떨어지는 벚꽃 애니메이션 효과

#### 2. Greeting Messages
- 하객 그룹별 맞춤 메시지

#### 3. Photo Slideshow
- 약 30장의 웨딩 사진 슬라이드쇼

#### 4. Making Film
- 아내가 편집한 웨딩 비디오

#### 5. Calendar
- 손으로 그린 결혼식 카운트다운 타이머

#### 6. Navigation
- 네이버, Tmap, 카카오맵 연동

#### 7. Directions
- 지역별 상세 교통 안내

#### 8. Ceremony Information
- 식사 장소, 시간, 주차 정보

#### 9. Congratulations Messages
- 하객 방명록 기능
- 축하 메시지 보존

#### 10. Thanks & Sharing
- 맞춤 감사 메시지
- 카카오톡 공유
- 링크 공유 기능
