# 분기별 리뷰

## 체크리스트

### 1. 전체 문서 구조 점검
- [ ] 문서 구조가 최신 상태인가?
- [ ] 불필요한 파일이 있는가?

### 2. 오래된 프로젝트 간소화
- [ ] 3년 이상 된 프로젝트가 과도하게 상세한가?
- [ ] 핵심 성과만 남기고 간소화할 부분이 있는가?

### 3. Professional Summary 업데이트
- [ ] 현재 커리어 방향과 일치하는가?
- [ ] 7년차 시니어 강점이 명확한가?

### 4. 템플릿 디자인 검토
- [ ] 다크모드 정상 동작
- [ ] 모바일 반응형 정상
- [ ] CSS 변수 일관성
